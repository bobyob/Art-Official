# Art-Official

MS Paint Clone project made with HTML, JavaScript, and CSS. 

Features include basic MS Paint features - brush, pen, pencil, colours, background, fill, shapes, erase

Other features: Undo/redo, download + download file type specification

Improvements to MS Paint: an AI chatbot option that views your screen and provides tips and improvements

Future implementation ideas: AI Drawing companion for step-by-step drawing instructions through

The project was built and deployed within a 5-hour time limit
